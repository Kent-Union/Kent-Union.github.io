    Creating a Membership Product

    Setting up the memership under Activities > Management

    1. Name: When naming a product try to follow a consistent convention:

      {Activity Name} Standard Membership

      {Activity Name} Associate Membership

      {Activity Name} Social/Team/Development etc. Membership

      Warning

      Always include the activity name

    2. Report: The Report Name is usually always identical to the above, just capitalised. This is what will appear on the receipt

      Note

      Bear in mind the Report Name can only be 40 characters long so you may need to abbreviate it slightly e.g. ASSOCIATE into ASSOC

    3. Price: Enter the price of the membership

    4. Inventory: Inventory is if you wanted to limit memberships to a particular amount for some reason. If you’re unsure, just set to 9999

    5. Start Date/End Date: Set the window of time you'd like the membership on sale for

      For normal year-long memberships, set the End Date to the same as the Subscription End Date which is 31-07-2020 23:59. This ensures students won’t be able to buy products in the summer break

    6. Subscription End Date: This is the date when the student’s membership to a club/society expires

      For normal year-long memberships, set this to 31-07-2020 23:59

    7. Department/Nominal: Set to the associated department and nominal

    8. VAT: Set to T8 – Student Income – OoS

    9. Period: Set to 0

    10. Platinum Product and Collect Guest Details can be left unchecked

    11. Online Sales: Ensure this box is checked. This allows the membership to be sold online the Kent Union shop page under the relative activity group

    12. Offline Sales: Ensure this box is checked. This allows the product to be sold at Mandela Reception

      Note

      Note that this also requires extra work adding the product to the Mandela till menu

    13. Door Entry Sales: Leave this unchecked

    14. Max Per Transaction: Set to 1

    15. Assign Purchase to Individual: Leave this box unchecked

    16. Rules: If you’re creating an Associate Membership be sure to select ‘Associate Member’ on the dropdown and click Add

    17. Member Restriction: If you want to ban a particular student from purchasing the product you can search by student ID

    Completing the membership under EPOS

    After you’ve created the membership product under Activities > Management, you’ll need to go to the EPOS products module in order to finish it.

    1. Go to EPOS > Zones > Reception > Enter Zone > Products
    2. Click Filter and search for the product you just created
    3. Click the blue Modify button and Expand All

    Under Basic Information

    1. Name and Caption: These should be prepopulated

    2. URL Name: Here you can alter the URL of the product online (e.g. /shop/product/{URL-name})

      Info

      If left blank it’ll default to the product ID on the URL

    3. Barcode: Leave blank

    4. Description: Enter a description to appear on the product page
    5. Category: Set to None
    6. Active: Ensure this box is checked
    7. Fixed Price: Ensure this box is checked
    8. Full Price: This should be prepopulated with the price previously entered

    Under Shop Options

    1. Online/Offline Sales: This should be prepopulated
    2. Member Sales: Ensure this is checked. This forces the student or associate member to be signed into the website to purchase the product
    3. Public Sales Leave this box unchecked. This should rarely be used as it allows someone to buy a product without signing into the website which means the transaction won’t be tied to any account
    4. Age Check: Leave this box unchecked
    5. Booking Fee: Set this box to unchecked

    6. Product Image: If you’d like the product to have an image attached to it on the website (e.g. a group logo for membership products or a photo of the sports kit) upload it here

      Warning

      Make sure to follow the file constraints underneath

    7. Available Categories: Ignore this section

    Under Shop Rules

    1. Max Per Transaction: Set to 1
    2. Max Per Person: Set to 1
    3. Available Rules: Another way of adding an Associate Member rule to Associate Memberships

    Under Additional Options

    1. App Section: Leave as None
    2. Activity: Allows you to limit a product to be purchased a specfic activity group only (e.g. trips)
    3. Subscription Type and Subscription End Date: These should be prepopulated

    Under Extra Capture Fields

    1. Available Fields: Used if you want to capture extra information on a product and not really applicable to memberships

      Note

      These are more suited to products such as event tickets for seating arrangements or meal choices
